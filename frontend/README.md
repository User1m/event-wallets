## Description

Starter Frontend App

## Technologies
- Typescript
- Graphql
- React & [CRACO](https://github.com/dilanx/craco)

## Pre-requisites
- [Node](https://nodejs.org/en/download/)

## Installation

```bash
> yarn ci
```

## Running the app

```bash
RUN
> export GRAPHQL_URL=https://api.[develop|staging].usespritz.com/graphql
> yarn schema:gen:ci   
> yarn start 

```

## Test

```bash
# unit tests
> yarn test
# e2e tests
> yarn cypress:open
```
